Välkommen!

Den här hemsidan handlar om olika svampar som man kan hitta i skogarna i vårt avlånga land. Det är ett projekt i Webbutveckling 1 som använder Bootstrap. Om du kikar runt på hemsidan så kan du hitta flera grejer. Du kan läsa mer om de delarna här nedan eller genom att klicka dig in i menyn.

Svamplista

På sidan som listar olika svampar kan du se svampar sorterade i olika kategorier. Det finns uppgifter om hur goda de anses vara, vart man kan hitta de, samt om det finns några förväxlingssvampar. På vissa ställen finns även mina egna kommentarer om svamparna.

Letarguide

Här hittar du några tips om hur du kan börja med att leta efter, preparerar, och tillagar svamp.

Svampkalender

Här finns en kalender som visar ungefär när på året man kan hitta olika svampar. Här kanske jag även nämner lite gammal skrock för att hjälpa dig i ditt letande.
